## ADDED Requirements

### Requirement: 系统托盘图标
应用 SHALL 在系统托盘显示常驻图标，提供快速访问入口。

#### Scenario: 启动时显示托盘图标
- **WHEN** 应用启动
- **THEN** 系统托盘显示 PaseBoard 图标

#### Scenario: 点击托盘图标显示主窗口
- **WHEN** 用户点击系统托盘图标
- **THEN** 应用显示主窗口（如果已隐藏则恢复显示）

### Requirement: 设备列表界面
UI SHALL 显示设备列表，包括设备名称、连接状态、IP 地址。

#### Scenario: 显示在线设备列表
- **WHEN** 用户打开主窗口
- **THEN** UI 显示所有已发现设备，在线设备显示绿色指示器

#### Scenario: 显示离线设备
- **WHEN** 设备离线
- **THEN** UI 将该设备显示为灰色，状态标记为"离线"

### Requirement: 历史记录界面
UI SHALL 提供历史记录查看界面，显示最近的粘贴板内容。

#### Scenario: 加载历史记录列表
- **WHEN** 用户打开历史记录标签页
- **THEN** UI 显示最近 100 条粘贴板记录，按时间倒序排列

#### Scenario: 点击历史记录复制到粘贴板
- **WHEN** 用户点击某条历史记录
- **THEN** 系统将该记录的内容复制到粘贴板

### Requirement: 历史记录显示格式
每条历史记录 SHALL 显示内容预览、来源设备、时间戳。

#### Scenario: 内容预览截断显示
- **WHEN** 历史记录内容超过 100 字符
- **THEN** UI 显示前 100 字符并添加省略号"..."

#### Scenario: 显示相对时间
- **WHEN** 显示历史记录时间戳
- **THEN** UI 显示相对时间（如"2 分钟前"、"1 小时前"）

### Requirement: 托盘菜单
系统托盘图标 SHALL 提供右键菜单，包含"显示主窗口"和"退出"选项。

#### Scenario: 右键显示托盘菜单
- **WHEN** 用户右键点击托盘图标
- **THEN** 显示包含"显示主窗口"和"退出"的菜单

#### Scenario: 点击退出关闭应用
- **WHEN** 用户点击托盘菜单中的"退出"
- **THEN** 应用完全关闭，托盘图标消失

### Requirement: 窗口最小化到托盘
应用 SHALL 在用户关闭主窗口时最小化到托盘，而非完全退出。

#### Scenario: 关闭窗口最小化到托盘
- **WHEN** 用户点击主窗口的关闭按钮
- **THEN** 主窗口隐藏，应用继续在后台运行，托盘图标保持显示

#### Scenario: 完全退出仅通过托盘菜单
- **WHEN** 用户想完全退出应用
- **THEN** 必须通过托盘菜单点击"退出"

### Requirement: 警告提示
UI SHALL 在内容超过大小限制时显示警告提示。

#### Scenario: 内容超过 10MB 时显示警告
- **WHEN** 粘贴板内容超过 10MB
- **THEN** UI 显示通知："内容超过 10MB，不同步"

#### Scenario: 警告自动消失
- **WHEN** 警告显示 3 秒后
- **THEN** 警告自动消失
